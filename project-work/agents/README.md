# Agents 目录说明

这里存放 `DorisClaw` 第一轮基建的 agent 独立工作目录。

当前目录：

- `agent-a-data-foundation`：数据基础层
- `agent-b-query-intelligence`：查询智能层
- `agent-c-insight-forecast`：洞察与预测层
- `agent-d-frontend-workbench`：前端工作台层
- `agent-e-cli-mcp-runtime`：CLI / MCP / 运行时层

- `README.md`
- `orders/`
- `tickets/`
- `reviews/passed/`
- `reviews/rework/`
