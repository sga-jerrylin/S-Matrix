# E-order-20260422-1329-006-Doris4.0.5运行时升级指令单

## 1. 指令主题

将本地 Doris 运行时从 `4.0.0` 升级到 `4.0.5`，并保证 Docker / runtime / smoke / 数据源保存链路继续可用。

## 2. 为什么由你负责

本次升级属于运行时层与本地联调环境层，不是业务逻辑升级。

你持续 owner 的范围已经覆盖：

1. `docker-compose.yml`
2. `init.ps1` / `init.sh`
3. `update.ps1` / `update.sh`
4. `doris-api/dc_runtime/doctor.py`
5. `scripts/smoke_docker.sh`
6. 本地 Docker / Doris 启停、健康检查、恢复路径

所以这轮继续由 `agent-E` 做，不新开 agent。

## 3. 本次目标

本次只做 Doris patch line 升级，不做 Doris 大版本路线调整。

目标：

1. `apache/doris:fe-4.0.0` -> `apache/doris:fe-4.0.5`
2. `apache/doris:be-4.0.0` -> `apache/doris:be-4.0.5`
3. 维持当前单 backend 拓扑：`smatrix-be:9050`
4. 升级后 `dc smoke`、系统表写探针、公开保存数据源 API 仍通过
